FORMeGO

FORMeGO je browser extension pre rýchle a pohodlné vypĺňanie formulárov pomocou uložených profilov. Používateľ si vie uložiť viac kont, odomknúť šifrovaný vault a jedným klikom doplniť meno, kontakt, adresu aj ďalšie údaje do webového formulára.

Čo aplikácia vie

Hlavné funkcionality

👤 Viac profilov

Ukladanie viacerých kont, napr. Ja, sestra, tato alebo pracovný profil.

📝 Vyplnenie na jeden klik

Po výbere profilu doplní relevantné polia v rámci formulára automaticky.

🔒 Ochrana údajov

Profily sú uložené v šifrovanom vaulte a pri práci sa odomykajú heslom.

📍 Heuristické mapovanie

Rozpoznáva meno, email, telefón, adresu, PSČ, mesto, firmu a ďalšie polia.

🧠 Kompatibilita s modernými webmi

Používa native value setter a eventy, aby fungoval aj na React alebo Vue formulároch.

🛠 Jednoduchá lokálna inštalácia

Rozšírenie sa dá spustiť cez Developer mode bez publikovania do Chrome Web Store.

Stiahnutie a spustenie

Ako rozšírenie nainštalovať

🧩 Chrome / Edge

Projekt sa dá stiahnuť ako ZIP a otvoriť ako lokálne rozšírenie v Developer mode.

Stiahnuť ZIP projektu

📋 Návod na inštaláciu

  1. Stiahni alebo naklonuj repozitár.
  2. Otvor chrome://extensions alebo edge://extensions.
  3. Zapni Developer mode.
  4. Klikni na Load unpacked.
  5. Vyber priečinok autofill-profiles.
