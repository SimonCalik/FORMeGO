FORMeGO

FORMeGO je rozšírenie pre prehliadač, ktoré zrýchľuje vypĺňanie formulárov pomocou uložených profilov. Hodí sa pri registráciách, objednávkach a opakovanom zadávaní osobných alebo kontaktných údajov.

Čo aplikácia vie

Hlavné funkcionality

👤 Viac profilov

Uloženie viacerých kont ako Ja, sestra, tato alebo pracovný profil.

📝 Vyplnenie formulára na jeden klik

Po výbere profilu doplní relevantné polia v rámci formulára automaticky.

🔐 Šifrovaný vault

Citlivé údaje sa držia v šifrovanom uloženisku a pri práci sa odomykajú heslom.

📍 Heuristické mapovanie polí

Rozpozná meno, e-mail, telefón, adresu, mesto, PSČ, firmu a ďalšie polia.

🧠 Reakcia na moderné formuláre

Používa native value setter a input/change eventy kvôli React a Vue kompatibilite.

🛠 Developer mode inštalácia

Rozšírenie sa dá nahrať lokálne cez Chrome alebo Edge bez publikovania v store.

📋 Návod

  1. Stiahni alebo naklonuj repozitár.
  2. Otvor chrome://extensions alebo edge://extensions.
  3. Zapni Developer mode.
  4. Klikni na Load unpacked.
  5. Vyber priečinok autofill-profiles.
